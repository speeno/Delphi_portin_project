## 1. 한 줄 요약

- **총판 전용**: 기초관리에 **출판사관리** 메뉴가 있음.
- **출판 전용**: 기초관리에 **메뉴보이기·메뉴감추기**가 있고, 최상위에 **년/월(통계)** 메뉴 전체가 추가됨.
- **공통**: **거래관리·원장관리·회계관리·자료관리·통계관리** 다섯 구역은 두 빌드 모두 **하위 캡션 나열이 동일**(순서·항목 일치).
- **기초관리**: 공통 항목이 있으나 **표시 순서가 다르고**, 위 전용 항목 유무가 다름.

| 구분 | 도서유통-총판 | 도서유통-출판 |
|------|----------------|----------------|
| 기초관리 | **출판사관리** 포함 | 출판사관리 **없음** |
| 기초관리 | 메뉴보이기·메뉴감추기 **없음** | **메뉴보이기**, **메뉴감추기** 포함 |
| 최상위 | 년/월 통계 메뉴 **없음** | **년/월(통계)** 및 아래 하위 항목 전부 |

## 6. 동일 구역(검증 결과)

아래 최상위 메뉴 아래의 **모든 메뉴 캡션(플랫 순서)** 을 두 파일에서 비교했을 때 **완전히 동일**했음.

- 거래관리  
- 원장관리  
- 회계관리  
- 자료관리  
- 통계관리  

즉, 위 다섯 영역의 화면 진입점 이름·구분선 위치는 두 프로그램이 같고, 차이는 **기초관리**와 출판 측 **년/월(통계)** 에 집중됨.

## 7. 재현 방법 (감사용)

동일 결과를 내려면 `MainMenu1` 블록에서 `Caption = '...'` 만 계층 들여쓰기와 함께 추출하면 됨. 디코딩은 **CP949** 기준.
