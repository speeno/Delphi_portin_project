### Generate `14-*` files locally (schema + cross-ref)

Do **not** commit passwords. Use env vars or `--password-file` (single-line UTF-8 file, chmod 600, outside the repo).

```bash
pip install -r debug/requirements-db.txt
export MYSQL_USER='root'
export MYSQL_PASSWORD='...'   # or omit and use --password-file
export MYSQL_HOSTS='115.68.3.153,115.68.3.154,115.68.3.155,115.68.7.138'
# 구형 MySQL에서 (1115) Unknown character set: utf8mb4 이면:
# export MYSQL_CHARSET=utf8
python3 debug/check_mysql_hosts.py
python3 debug/check_mysql_hosts.py --export --database '<your_schema>' --export-on-host 115.68.3.153
# 위 export 성공 시 `14-db-code-vs-live.md`는 기본으로 자동 갱신됨. 수동만: `python3 debug/check_mysql_hosts.py --crossref-only`
python3 debug/generate_db_porting_report.py
pip install -r debug/requirements-charts.txt   # optional
python3 debug/generate_phase1_charts.py
python3 debug/report_wireframe_coverage_gap.py
```

If `MYSQL_DATABASE` is unset, the export step prints **non-system databases**; pick the real schema name and re-run with `--database`.
